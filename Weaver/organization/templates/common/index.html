Molecular biology and administration tools

Explore weaver highlights

Weaver contains several tools that allow researcher to:

  • Simple and rapid - golden gate based - in silico plasmid assembly wizard
  • Level 0 part designer
  • Plasmid restriction enzyme digestion design
  • PCR design
  • Online sequence manipulation
  • Sequence file conversion / download tool
  • Database blast

Weaver implements Teselagen's Open Vector Editor.

This tool provides:

  • Circular / linear DNA sequence inspection
  • Restriction enzyme cut prediction and digestion simulation
  • Online DNA sequence edition
  • DNA sequence translation

Weaver provides administration tools to:

  • Manage plasmids, primers, glycerol stocks (boxes), restriction enzymes and more
  • Collaborative cloud work
  • Organize all elements in projects
  • Print samples labels
